# FocusMind

Aplicación de enfoque y productividad personal.

## Inicio rápido

Consultá [SPEC.md](./SPEC.md) para la documentación técnica completa.

### Backend

```bash
cd backend
cp .env.example .env   # Linux/macOS — en Windows: copy .env.example .env
npm run dev
```

### Frontend

```bash
cd frontend
cp .env.example .env   # Linux/macOS — en Windows: copy .env.example .env
npm run dev
```
